# Datasets & Sources

## Purpose
List and describe datasets needed and collected for hydrodynamic and structural modelling in Project TMS.

## Required Datasets
### 1. Bathymetry Data
- Sources: NOAA, GEBCO, regional datasets
- Purpose: Realistic wave propagation modelling

### 2. Historical Tsunami Records
- Arrival times, wave heights, locations
- Use case: Model validation

### 3. Structural Material Properties
- Steel, concrete, composite materials
- Elastic modulus, density, yield strength

### 4. Coastal Infrastructure Layouts
- Building height maps
- Shoreline geometry

## Data Format Notes
- Prefer CSV, NetCDF, or raw gridded data.
- Large datasets kept in `/data/`.
